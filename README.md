# AttaurRahman Portfolio (AGI)

##  Overview

This is a simple personal portfolio webpage built using **HTML** and **CSS**. It showcases a navigation bar, a hero section with profile information, and a responsive card layout.

---

##  Features

* Responsive **navigation bar**
* Clean **hero section** with centered profile image and text
* Flexible **card layout** using Flexbox
* Simple and modern UI design
* Basic hover effects

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox & Media Queries)

---

##  Project Structure

```
project/
│
├── index.html
├── css/
│   └── style.css
├── img/
│   └── profile.jpg
├── images/
│   └── (card images)
```

---

##  How It Works

* The **Navbar** contains navigation links.
* The **Hero Section** displays your profile image inside a circle and text centered inside a card.
* The **Cards Section** uses Flexbox to create a responsive grid layout.
* The **Footer** displays copyright information.

---

##  Responsiveness

* Uses Flexbox and media queries.
* Navbar stacks on smaller screens.
* Cards wrap automatically to fit screen size.

---

##  Setup Instructions

1. Clone or download the project.
2. Ensure your folder structure matches the one above.
3. Open `index.html` in your browser.

---

##  Customization

* Replace `/img/profile.jpg` with your own image.
* Update text content in the hero section.
* Modify colors in `style.css` to match your branding.

---

##  Notes

* Keep image paths consistent (`/img` vs `/images`).
* Avoid fixed widths like `2026px` for better responsiveness.
* Use `object-fit: cover` for clean image display.
