

This project is a **frontend practice layout** featuring a navigation bar and a hero section.  
It is built using **HTML and CSS** to improve understanding of layout structure, semantics, and basic UI design.
- Simple navigation bar with logo and links
- Hero section with:
  - Badge/title
  - Name heading
  - Short description
  - Call-to-action buttons
  - Profile image
- Clean and readable HTML structure
- Beginner-friendly codebase
- **HTML5** – page structure and semantics  
- **CSS3** – layout, styling, and responsiveness (to be expanded)
